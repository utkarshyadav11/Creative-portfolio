## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/utkarshyadav11/Creative-portfolio.git
   cd Creative-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## âš™ï¸ Configuration

The entire portfolio is data-driven, making it easy to update without touching complex code.

- **Personal Info**: Update `src/resources/content.tsx` to change your name, role, social links, and skills.
- **Design System**: Update `src/resources/once-ui.config.ts` to change brand colors, fonts, and global styles.
- **Projects**: Add new `.mdx` files to `src/app/work/projects/`.
- **Blog**: Add new `.mdx` files to `src/app/blog/posts/`.
